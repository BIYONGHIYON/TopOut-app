# TopOut

**등반 시작부터 완등 후보 순간까지, 필요한 구간을 자동으로 찾아주는 클라이밍 촬영 앱**

`iOS` · `Android`

<sub>현재 개발 버전: 1.4 · build 5</sub>
</div>

---

## 소개

**TopOut**은 실내 클라이머가 긴 촬영 영상에서 실제 등반 구간을 더 빠르게 찾고 저장할 수 있도록 돕는 모바일 앱입니다.

사용자가 촬영 구도와 매트 경계선을 설정하면 앱이 매트 주변의 영상 변화와 사람의 자세를 단계적으로 분석해 등반 시작과 종료 후보 시점을 찾습니다.

- 탑홀드를 선택한 경우: 선택한 홀드와 신체 관절이 가장 가까워진 순간을 종료 후보로 사용
- 탑홀드를 선택하지 않은 경우: 등반 중 신체가 가장 높은 위치에 도달한 순간을 종료 후보로 사용

녹화는 구간이 감지되어도 자동으로 멈추지 않습니다. 사용자가 종료 버튼을 누를 때까지 하나의 원본 영상으로 계속 기록하며, 한 번의 촬영 안에서 여러 등반 시도를 각각 감지합니다. 촬영이 끝난 뒤 자동 제안 구간을 확인하고 시작점과 종료점을 직접 조정한 다음 필요한 부분만 저장할 수 있습니다.

## 주요 기능

### 매트 경계선 자동 탐지 및 수동 보정

MobileSAM을 이용해 벽과 매트가 만나는 경계를 자동으로 찾습니다. 탐지 결과가 실제 환경과 다르면 화면의 조절점을 드래그해 직접 수정할 수 있습니다.

- 완만하게 기울어진 매트 경계 지원
- 관절의 가로 위치에 맞는 경계 높이를 계산해 시작·종료 판단
- 자동 탐지된 경계를 별도의 수직 이동 없이 원래 위치 그대로 사용
- 자동 탐지 실패 시 기본 경계 제공

### Motion Detection → Pose Detection 단계형 분석

사람이 없는 동안에는 비교적 가벼운 움직임 분석을 사용하고, 의미 있는 움직임이 감지된 뒤에 자세 분석으로 전환합니다. 이를 통해 불필요한 추론을 줄이면서 등반 시작과 종료 후보를 추적합니다.

분석 빈도는 상황에 따라 조절됩니다.

- 초기 움직임 탐색과 정상 등반 추적: 저빈도 분석
- 등반 시작 전 자세 추적과 일시적인 자세 유실 복구: 고빈도 분석

### 온디바이스 자세 분석

자세 분석은 사용자의 기기 안에서 수행됩니다.

- **iOS:** Apple Vision Human Body Pose
- **Android:** MoveNet SinglePose Thunder FP16 · LiteRT

촬영 영상과 분석용 이미지 프레임은 TopOut 개발자 서버로 전송되지 않습니다.

### MobileSAM 기반 탑홀드 선택

목표 홀드를 화면에서 탭해 선택할 수 있으며, MobileSAM이 선택 영역을 분할해 마스크와 외곽선으로 표시합니다.

- **iOS:** Core ML MobileSAM
- **Android:** ONNX Runtime MobileSAM

### 자동 등반 구간 감지

매트 경계와 자세 변화를 바탕으로 등반 시작과 종료 후보 시점을 계산합니다.

- 시작: 가장 낮게 검출된 주요 관절이 시작 경계 위로 올라온 순간
- 종료 확인: 가장 낮은 주요 관절이 종료 경계 아래로 다시 내려온 순간
- 저장 후보: 탑홀드 최접근 순간 또는 등반 중 최고 도달 순간
- 시작 경계: iOS는 매트선보다 화면 높이 6% 위, Android는 4% 위
- 종료 경계: 두 플랫폼 모두 매트선보다 화면 높이 2% 위
- 등반 중 자세 유실: 두 플랫폼 모두 최대 6초 동안 복구 시도

자동 감지 결과는 최종 판정이 아니라 **편집 후보 구간을 제안하기 위한 보조 정보**입니다.

### 다중 시도 감지

완등 후보 시점이 확인되어도 녹화는 계속됩니다. 한 번의 연속 촬영 안에서 여러 등반 시도를 감지하며, 유효한 각 시도는 서로 가까이 발생하더라도 별도 구간으로 유지합니다.

짧은 오탐 후보만 주변 구간과의 관계를 확인해 병합하거나 제외한 뒤, 각 구간에 편집용 여백을 적용합니다.

### 자동 선택 및 정밀 트리밍

촬영 종료 후 감지된 등반 구간을 자동으로 제안합니다.

- 감지된 여러 시도 간 전환
- 시작점·종료점 수동 조정
- 느린 드래그를 통한 정밀 조정
- 탑홀드 외곽선 포함 여부 선택
- 선택 구간만 저장

### 녹화 중 실시간 화면 OFF

기본 설정에서는 녹화 중 실시간 카메라 화면을 표시하지 않습니다.

카메라 센서, 영상 녹화, 움직임 분석 및 자세 분석은 계속 동작하지만 프리뷰 렌더링을 중단해 장시간 촬영 시 화면 표시로 인한 배터리 소모와 발열을 줄입니다. 촬영 중 실시간 화면을 다시 켤 수도 있습니다.

### 안전한 녹화 종료

다음 상황에서는 현재까지 촬영된 영상이 손상되지 않도록 녹화를 안전하게 마무리합니다.

- 앱이 백그라운드로 이동한 경우
- 전화 수신 또는 다른 앱의 카메라·마이크 사용으로 세션이 중단된 경우
- 카메라 세션 오류가 발생한 경우
- 남은 저장공간이 안전 기준보다 부족한 경우

## 작동 흐름

```text
촬영 옵션 설정
→ 구도 및 매트 경계 확인
→ 선택적으로 탑홀드 지정
→ 연속 녹화 시작
→ 매트 위쪽 움직임 탐색
→ 기기 내 자세 분석
→ 등반 시작·완등 후보·복귀 시점 기록
→ 여러 등반 구간 제안
→ 시작점·종료점 정밀 조정
→ 사진 또는 동영상 보관함에 저장
```

## 기본 촬영 설정

| 항목 | 기본값 |
|---|---|
| 해상도 | FHD |
| 프레임률 | 60fps |
| 화면 비율 | 9:16 |
| 자동 구간 탐지 | ON |
| 탑홀드 마스킹 | OFF |

지원 후보:

- 해상도: FHD, 4K
- 프레임률: 30fps, 60fps
- 화면 비율: 9:16, 4:5, 3:4, 1:1
- 세로·가로 촬영

실제 선택 가능한 해상도와 프레임 조합은 기기의 후면 카메라 성능에 따라 달라질 수 있습니다.

## 개인정보 및 영상 처리

- TopOut은 회원가입 없이 사용할 수 있습니다.
- 이름, 이메일 주소, 위치정보를 앱 기능 수행 목적으로 수집하지 않습니다.
- 촬영한 원본 영상과 분석용 이미지 프레임을 TopOut 개발자 서버로 업로드하지 않습니다.
- 움직임 감지, 자세 추정, 매트 경계 탐지와 탑홀드 분할은 사용자의 기기 안에서 수행됩니다.
- 사용자가 저장을 선택한 결과 영상만 기기의 사진 또는 동영상 보관함으로 내보냅니다.

자세한 내용은 개인정보 처리방침에서 확인할 수 있습니다.
